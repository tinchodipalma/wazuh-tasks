## Task6: React App

Web page that shows a list of vehicles and brands.

#### Requirements

- `node.js v12.x.x` or highter

#### Instructions

1. Install dependencies: `npm install`
2. Execute `npm run start`
3. Go to http://localhost:3000/

#### Tests

1. Execute `npm run test`
