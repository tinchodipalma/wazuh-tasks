## Task4: Node.js

Build a simple Node.js server using Hapi.js, it must include a GET endpoint that returns
an object like this:

```
{
  message: "Hello world"
}
```

#### Requirements

- `node.js v12.x.x` or highter

#### Instructions

1. Install dependencies: `npm install`
2. Execute `npm run start`
3. Go to http://localhost:3000/
